# STAR SMASH! 💋


> *The tabloid that tells it like it iz.*

A Y2K-themed interactive tabloid magazine — built from scratch as a multidisciplinary design portfolio project.

🌐 **Live site:** https://starsmash.netlify.app

---

## ✨ The concept

Most student portfolios are an "About Me" page with thumbnails. This one *is* the project.

STAR SMASH! is a fictional 2000s gossip magazine covering fictional celebrities who don't exist. Every page is a designed magazine spread — the website itself is the work sample, the editorial layout, the brand identity, and the interaction design all at once.

The twist: the writing is gossip-column nonsense, but the content underneath is real. The "scandal" article hides a data visualization. The Met Gala "ranking" is built on actual color theory. The "About the Editor" page is a real CV in disguise.

---

## 📄 Pages

| Page | What it is | What it shows |
|------|-----------|---------------|
| `index.html` | The magazine cover — animated, sticker-covered, classic Y2K | Layout, hierarchy, CSS animation, typography |
| `article-1.html` | "EXPOSED: The Spotify Genre QUIETLY Dying" | Editorial layout, data visualization (Chart.js) |
| `article-2.html` | "Met Gala RANKED by Color Theory" | Comparative ranking layout, palette design, design vocabulary |
| `quiz.html` | "Which Tragic Pop Star R U?" personality quiz | Vanilla JavaScript: state, event handling, conditional logic, DOM manipulation |
| `about.html` | "WHO IS SHE?" — a tabloid exposé about the editor (me) | Real CV + contact info disguised as gossip |

---

## 🛠 Built with

- **HTML5** — semantic markup
- **CSS3** — gradients, `clip-path`, flexbox, grid, keyframe animations, custom properties
- **Vanilla JavaScript** — no frameworks; quiz logic, DOM manipulation, event listeners
- **Chart.js** — interactive data visualization
- **Google Fonts** — Bungee + Comic Neue for cross-device consistency

No frameworks, no build tools, no dependencies beyond Chart.js. Plain HTML/CSS/JS that runs anywhere.
